# 微服务框架
## Java
### Spring Cloud
  Spring Boot的封装
  Spring Config 增加了一部分分布式系统的基础设施实现
### Dubbo
  远程服务注册、调用、管理的分布式框架。（RPC）
  
## Go  
### Go Kit(gokit.io)
  传输层、接口层、服务层
### Go Micro
  插件化，RPC服务框架。
  [插件仓库](https://github.com/micro/go-plugins)

# 领域驱动设计
  路径依赖法则：与惯性性质一样，一旦进入某一路径就可能对这种路径产生依赖。从单体架构到分布式架构，路径思路要改变。
