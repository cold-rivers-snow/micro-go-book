# 四层IT系统
## 传统IT
## IaaS
   网络、存储、服务器、虚拟化
## PaaS
   IaaS + 操作系统、中间件、运行
## SaaS
   PaaS + 数据、应用程序

# 虚拟化与容器化区别
  虚拟化技术代表VM虚拟机，从操作系统层面隔离开应用程序；共享硬件资源。
  容器化代表docker，从软件层面分离，共享操作系统。

  K8s用来管理调度Docker集群，可以实现自动化部署。可以将docker当成k8s内部的低级组件。k8s不仅支持docker，还支持Rocket等其他容器技术。
